Se você é um membro do público em geral:

O fato de que você está vendo esta página indica que o site que você acabou de visitar ou é problema ou está passando por manutenção de rotina.

Por exemplo, se você teve problemas ao visitar .andreeu.net, você deve enviar e-mail para "suporte@andreeu.net".

Sobre o CentOS:

A Comunidade sistema operacional corporativo (CentOS) é uma empresa de classe Distribuição Linux derivada de fontes gratuitamente fornecidas ao público por um norte-americano de destaque da empresa fornecedor Linux. CentOS seja plenamente compatível com a política de redistribuição de fornecedores a montante e pretende ser binário 100% compatível. (CentOS muda principalmente pacotes para remover a marca do fornecedor a montante ea arte.) O Projeto CentOS é a organização que constrói CentOS.

Para obter informações sobre CentOS por favor visite o website CentOS.

Nota:

CentOS é um sistema operativo e é utilizado para alimentar este site, no entanto, o servidor web é propriedade do proprietário do domínio e não o projeto CentOS. Se você tiver problemas com o conteúdo deste site, entre em contato o dono do domínio não, o projeto CentOS.

A menos que este servidor está no domínio CentOS.org, o Projeto CentOS não tem nada a ver com o conteúdo deste ou de qualquer servidor web e-mails que você dirigiu a este site.
